1 What does AGI stand for, and difference between AI and AGI?

AGI = Artificial General Intelligence.

  • AI: solves specific tasks (narrow). Example: translation, face detection.
  • AGI: can learn and reason across many tasks like a human (general). Not achieved yet.
Key: AI = narrow, AGI = general
2 Challenges of AI and possible solutions

Challenges

  • Bad/biased data
  • Black-box models (hard to explain)
  • High compute cost
  • Privacy and ethics risks
  • Deployment + maintenance issues

Solutions

  • Improve data quality, reduce bias, good labeling
  • Use explainable AI (XAI)
  • Optimize models + use efficient infrastructure
  • Apply governance, privacy, ethics rules
  • Monitor models, retrain regularly
3 What does SWAT mean? What is an AI SWAT team and role?

SWAT = Special Weapons And Tactics.

AI SWAT team = a small expert group (AI/ML engineer, data engineer, product/domain expert) that rapidly builds AI solutions.

Role

  • Find high-impact use cases
  • Prototype fast (proof-of-concept)
  • Deploy and support AI adoption
4 ROI meaning, definition, and choosing the True North metric

ROI = Return on Investment.

Definition: measure of profit/value compared to cost.

Formula: ROI = (Gain − Cost) / Cost.

True North metric = one key business metric showing real value.

  • Must align with business goal (ex: revenue, cost savings, retention)
  • Must be measurable and trusted
  • Should reflect long-term success (not vanity metrics)
5 Common Big Data mistakes
  • Collecting too much irrelevant data
  • Poor quality: missing values, noise, duplicates
  • Ignoring bias and fairness
  • No clear data governance (ownership, rules)
  • Assuming “more data = better model” always
6 Software 1.0 vs Software 2.0
  • Software 1.0: humans write rules/code manually.
  • Software 2.0: models learn behavior from data (machine learning).
Software 2.0 = “data + training” instead of rules
7 Agile model vs Waterfall model

Waterfall

  • Step-by-step (requirements → design → build → test → release)
  • Hard to change once started
  • Testing mostly late

Agile

  • Iterative sprints (build small, test, improve)
  • Flexible requirements
  • Continuous feedback and testing
8 Evaluation metrics to compare predicted vs actual

Regression (numbers)

  • MAE: average absolute error
  • MSE: average squared error
  • RMSE: sqrt(MSE), interpretable scale
  • R²: how much variance is explained

Classification (labels)

  • Accuracy: correct / total
  • Precision: correct positives / predicted positives
  • Recall: correct positives / actual positives
  • F1: balance of precision and recall
  • ROC-AUC: ranking quality across thresholds
9 What is the Machine Intelligence Continuum?

A spectrum showing the growth of intelligent systems:

  • Rule-based automation
  • Machine learning
  • Deep learning
  • Reinforcement learning
  • AGI (future goal)
10 Which jobs will change as AI becomes advanced?

Many jobs will change (tasks get automated), especially:

  • Customer support, call centers
  • Data entry, admin work
  • Marketing and content workflows
  • Software development (coding assistance)
  • Healthcare documentation + triage support
  • Manufacturing / quality inspection
Most roles evolve: humans + AI
12 How can smaller / non-tech orgs compete for AI talent?
  • Offer meaningful impact and clear problems
  • Upskill existing employees
  • Use cloud AI tools / AutoML to reduce hiring pressure
  • Partner with universities, startups, consultants
  • Focus on domain-specific AI (your advantage)
13 Why collect the right data before training?
  • Models learn patterns only from data
  • Wrong data → wrong predictions
  • Good data reduces bias and improves accuracy
  • Saves time and cost (less rework)
Data quality > model complexity
14 What factors beyond algorithm decide ML success/failure?
  • Data quality, labeling, and coverage
  • Features + preprocessing
  • Domain knowledge / correct problem framing
  • Deployment environment and reliability
  • Monitoring (drift) + retraining plan
  • Business alignment and ROI measurement
15 Main opportunity for AI in enterprise functions right now
  • Automate repetitive processes (back office)
  • Decision support (analytics + forecasting)
  • Customer support assistants
  • Personalization (sales/marketing)
  • Efficiency + cost reduction
Biggest value now: “Augment people”
